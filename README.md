# Auction_CRUD
Backend server that implements a CRUD for a auction service including JWT for AUTH.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

The documentation is done using swagger, refer to /api for more information.
